== Description ==

Leyka the donations management system.

**Follow this plugin on [GitHub](https://github.com/Teplitsa/Leyka)**

More information at [leyka.te-st.ru](http://leyka.te-st.ru/).

Plugin been translated into the following languages:

1. Russian
2. English

**По русски**

Плагин «Онлайн-Лейка» – это плагин для системы управления содержанием WordPress, позволяющий собирать пожертвования для вашей организации или инициативы. 

**Следите за разработкой на [GitHub](https://github.com/Teplitsa/Leyka)**

Зная механизмы платежных систем (Яндекс.Деньги, Paypal и других), подготовив типовой текст договора оферты на пожертвование, мы создали инструмент для того, чтобы максимально сократить время подключения системы сбора пожертвований.

Для установки Лейки не требуется вносить изменения в код сайта, так как все можно сделать из административной консоли WordPress.  Для того, чтобы запустить механизм сбора денег на общественные цели достаточно установить один из  плагинов платежного шлюза, ввести свои реквизиты и оформить страницу цели пожертвования.

Основная аудитория – сотрудники НКО и общественные инициативы.

**Основные функции**

* Создавайте различные виды пожертвований для различных целей и проектов
* Отслеживайте приток пожертвований при помощи извещений на эл. почту
* Храните и отображайте на сайте короткие сообщения от доноров 
* Следите за статистикой пожертвований
* Собирайте платежи через разных платежных операторов
* Сами настройте текст вашей благодарности жертвующим
* Редактируйте текст оферты

== Installation ==

**In English**

1. Install and activate Easy Digital Download (Leyka base plugin).
2. Install and activate Leyka plugin.
3. Install needed payment gateways for Leyka.
4. Configure Leyka.
5. Configure payment gateways.
6. Create donation page.

For detailed setup instructions, visit the official [Documentation](http://leyka.te-st.ru/docs/install/) page.

**По русски**

1. Установить Easy Digital Downloads (база для работы Лейки).
2. Установить Лейку (база для работы Лейки).
3. Установить необходимые плагины платежных систем для Лейки.
4. Настроить Лейку.
5. Настроить плагины систем пожертвования.
6. Создать страницу пожертвования.

Для более подробной информации посетите страницу [документации](http://leyka.te-st.ru/docs/install/).
